# Snow Note
基于tauri+react+antd+react的笔记软件

## Why
* 简单、干净、易用、响应快
* 开源透明
* 支持全量导入导出

## 基本能力
1. 文档编辑
   1. 目录读取管理
   2. 文档编辑
   3. 文档保存
   4. 自动保存
   5. 新建
   6. 删除
   7. 移动
   8. 导入导出
2. 配置能力
   1. 主题
   2. 图片上传配置
   3. git仓库配置
3. git
   1. git日志分析回滚
   2. 自动git同步
   3.  修改历史同步

## 开发
1. yarn install
2. yarn run tauri dev
